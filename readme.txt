This is a simple web chat app created using HTML5, CSS, JAVASCRIPT, Nodejs
To run the web app:
1. In the terminal, open the project's directory, open the server.js file, and then in the terminal type "node server.js".
2. Go to the browser and go to localhost:3000 and the first thing to enter will be your name to identify your message from others.
3. Once you enter your name you can start chatting with anyone on the same localhost port.
